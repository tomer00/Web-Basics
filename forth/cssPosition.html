Css Positions

  1. Statc (Default)
  2. Relatuive
  3. Absolute
  4. Fixed
  5. Sticky
Static
- Default
- No use of left right top bootom properties

Relative
- Relative to parent
- We can use left right top props to add spacing around
- Other elements are no effefcted
- Initial space and layout is maintainded

Absolute
- Used more in overLay layouts
- Relative to closest positioned ancestor
- Space is removed by parent and other elements use the space

Fixed
- Used more in overLay layouts
- Positoned by parent but remain at its place if parent is scrolled
- Space is removed by parent and other elements use the space
