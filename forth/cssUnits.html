Css Sizes Units

    Absolute

    1. Pixel (px)
    2. Viewport width (vw)
    3. Viewport height (vh)

    Relative

    1. Percent (%)
    2. EM
    3. Relative em
Pixel
- Takes exact pixel size on window
- Absolute type and not depend on its parent

Percent
- Relative to parent
- Takes length according to imidiate parents property

Viewport Width
- Relative to viewports width
- Not depending on parent attributes

Viewport Height
- Relative to viewports Height
- Not depending on parent attributes
