Css Transition

CSS transitions allows you to change property values smoothly, over a given duration.

The Animatoin have these properties:

  • transition
  • transition-duration
  • transition-delay
  • transition-property
  • transition-timing-function

transition{ properties }

transition-duration : 4s; default is 0s;

transition-delay : 1s; Specify the delay for start of animation
default is 0s;
